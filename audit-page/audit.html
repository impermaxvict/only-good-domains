On this page you can audit all recorded domains. Blacklisted domains will be blocked and whitelisted domains allowed.

For each part of a domain you have three options (in the form of radio buttons):

  • RED = Blacklist this domain
  • AMBER = Same action as for the parent domain (This domain will not be added to any list!)
  • GREEN = Whitelist this domain

Click the ▶ buttons to show all recorded subdomains for a given domain.

If you want to create a whitelist or blacklist for another program, you should judge every domain on this page.

Do not forget to click the save button when done!
